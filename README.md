# lume-loading

Loading/transition animation slide.

## Preview

![Slide Preview](art/j-card.svg)

## Usage

Build the slide:

```bash
./build.sh
```

Produces `loading.vzglyd` — a packaged slide ready to be placed in your VZGLYD slides directory.

## Requirements

- Rust stable with `wasm32-wasip1` target: `rustup target add wasm32-wasip1`
